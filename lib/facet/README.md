## 💫 Integration with Scribble

Facet seamlessly integrates with the Scribble framework:
```rust
use facet::Calculator;

fn main() {
    let mut calc = Calculator::new();
    
    // Perform calculation
    let result = calc.compute("5 + 3")
        .with_coherence(0.95)
        .execute();
        
    println!("Result: {}", result);
}
```
